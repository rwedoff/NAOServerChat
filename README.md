# NAOServerChat
NAO Server Chat contains the server chat sofware that is used to run [NAO Remote](https://github.com/rwedoff/NAO-Remote) Android App found on [Google Play](https://play.google.com/store/apps/details?id=com.ryanwedoff.senor.naoservercontroller&hl=en).

##Installation 
Go to the releases tab and download NAOServerChat.zip.

####Windows
Click in the Windows Folder and use setup.exe to install.
#####Running
Open app, select the normal looking IP Address (E.g. 192.1.0.1) and type in a port.  By default, Port 8002 is open on Windows.  Other Ports might be blocked by Windows Firewall.

####Mac/Linux
Run Python Server Chat.py in terminal, the port is the only command line argument.

E.g.`python Python\ Server\ Chat.py 8002`
